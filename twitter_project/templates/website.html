Technology Stack

Built With Django

This project showcases the power and simplicity of Django, Python's high-level web framework that encourages rapid development and clean, pragmatic design.

  • Django Models for data structure
  • Django Templates for frontend
  • Django Views for handling requests
  • SQLite for database (configurable) 
# Sample Django model for tweets
class Tweet(models.Model):
    content = models.CharField(max_length=280)
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    created_at = models.DateTimeField(auto_now_add=True)
    likes = models.ManyToManyField(User, related_name='liked_tweets')
    
    def __str__(self):
        return f"{self.author.username}: {self.content[:30]}..."
        
    def like_count(self):
        return self.likes.count()

About The Project

This Twitter clone is a practice project created to explore and demonstrate Django's capabilities in building social media applications. It's designed as a learning exercise and portfolio showcase, not for professional or commercial use.

The project serves as a hands-on approach to understanding Django's core concepts including models, views, templates, user authentication, and database relationships. Perfect for developers looking to understand how social media platforms work under the hood.

"Built for educational purposes and skill development - showcasing the versatility of Django framework in creating interactive web applications."
